Install via Termux

Only Gemma 3 model is optimized for now.

    curl -fsSL -o install.sh https://android-artisan.github.io/AI-Droid/install.sh 
    bash install.sh
    ollama run Android_Artisan/Gemma3-Android

Requires Termux on Android or any Linux-based environment.
